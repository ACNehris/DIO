O que é HTML?

HTML é uma marcação usada para desenvolver projetos.

Por não ter a capacidade de criar funcionalidades ou softwares, não é considerada uma linguagem de programação.

Usada para construir websites, inserir e editar conteúdos. Além de desenvolver documentos eletrônicos para a internet.

O que são Atributos de Tag?

Atributos de Tag são marcações usadas para que o navegador reconheça o comportamento desejado pelo desenvolvedor.

Uma tag pode ter quantos atributos forem necessários.

Estrutura do HTML

O HTML é composto por 'HEAD' e 'BODY'.

Dentro de 'HEAD", são acrescentadas as informações que não serão vistas pelo usuário.

Já no 'BODY' vai todo o texto e informações que serão vistas e acessadas pelo usuário.
